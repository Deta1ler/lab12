Git Commands

Робота з локальним репозиторієм

Команда Опис
git init Створює новий локальний репозиторій
git status Перегляд стану файлів
git add . Додати всі зміни у staging area
git commit -m "message" Створення коміту

Робота з гілками

Команда Опис
git branch Показати всі гілки
git branch name Створити нову гілку
git checkout name Перейти на гілку
git merge name Об’єднання гілки з поточною

Робота з віддаленим репозиторієм (GitHub)

Команда Опис
git remote add origin URL Прив’язати локальний репозиторій до GitHub
git push -u origin branch Відправити гілку на сервер
git pull Отримати останні зміни з сервера
git clone URL Клонувати репозиторій
